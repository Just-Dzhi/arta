# installation

1. clone the repository:

    `git clone https://github.com/Just-Dzhi/arta.git`

2. install the dependencies:

    `npm install`

3. build the project:

    `npm run build`

4. edit .env file:

    `touch .env` and then replace values with your own.

5. run the bot:

    `npm run launch`
